# Task 1: Exploring and Visualizing the Iris Dataset

## Objective

The objective of this task is to explore and visualize the Iris dataset using Python. This helps in understanding data distributions, feature relationships, and basic data analysis techniques.

## Dataset Used

Iris Dataset (loaded using Seaborn)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Steps Performed

1. Loaded the Iris dataset.
2. Inspected the dataset using:

   * shape
   * columns
   * head()
   * info()
   * describe()
3. Checked for missing values.
4. Created visualizations:

   * Scatter Plot
   * Histograms
   * Box Plot
   * Correlation Heatmap
   * Pair Plot
5. Analyzed patterns and relationships among features.

## Key Findings

* Setosa species is clearly distinguishable from the other species.
* Petal Length and Petal Width have a strong positive correlation.
* The dataset contains very few outliers.
* Most features show a balanced distribution.

## Conclusion

The Iris dataset was successfully explored and visualized. Different visualization techniques helped identify feature relationships, distributions, and species patterns. This dataset provides a good foundation for learning data analysis and machine learning concepts.
